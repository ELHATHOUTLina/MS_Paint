# Paint Clone (React + Vite)

Projet front-end inspiré de l’ancien Microsoft Paint (style Windows 95/98), développé en React avec Vite.

L’objectif actuel est de reproduire l’interface visuelle rétro :
- barre de titre
- menu horizontal
- barre d’outils à gauche
- zone de dessin au centre
- palette de couleurs
- barre de statut

Pour le moment, le projet est un MVP visuel sans logique avancée de dessin.

## Démarrage rapide

1. Installer les dépendances
	npm install
2. Lancer le serveur de développement
	npm run dev
3. Ouvrir l’URL affichée dans le terminal

## Stack technique

- React
- Vite
- CSS (fichier principal : src/styles/paint.css)
